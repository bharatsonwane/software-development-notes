01. Data Types and Values

Primitives, objects, memory behavior, and practical type checks in JavaScript.

Primitives Objects References Type Checks

JavaScript is dynamically typed. A variable can hold any type, and type is tied to the value, not the variable.

let x = 10;      // number
x = "hello";     // now string
Two broad categories: primitives and objects (everything else).

1. Primitives

A primitive is a single, immutable value stored directly.

1.1 undefined vs null

let a;           // undefined
let b = null;   // intentionally empty

1.2 NaN and Infinity

typeof NaN;          // "number"
Number.isNaN(NaN);   // true
1 / 0;               // Infinity

1.3 Immutability

const s = "hello";
s.toUpperCase();   // "HELLO"
// s is still "hello"

2. Object Data Type

Objects are key-value collections. Arrays, functions, Date, Map, Set, and RegExp are objects too.

const person = { name: "Bharat", age: 30 };
const nums = [1, 2, 3];
typeof person;  // "object"
typeof nums;    // "object"
Array.isArray(nums); // true

4. Values and References

4.1 Primitive copy

let a = 10;
let b = a;
b = 20;
// a is 10, b is 20

4.2 Object reference copy

const obj1 = { x: 1 };
const obj2 = obj1;
obj2.x = 99;
// obj1.x is 99

4.3 Shallow vs deep copy

const original = { a: 1, nested: { b: 2 } };
const copy = { ...original };  // shallow
const deep = structuredClone(original);

5. typeof and instanceof

5.1 typeof quick checks

typeof "hello";      // "string"
typeof 42;           // "number"
typeof undefined;    // "undefined"
typeof null;         // "object" (legacy quirk)

5.2 instanceof

[] instanceof Array;      // true
[] instanceof Object;     // true
"hello" instanceof String; // false

7. Common Questions

  1. How many data types in JavaScript? 7 primitives plus object family.
  2. undefined vs null? Missing value vs intentional empty.
  3. Why is typeof null "object"? Historical compatibility behavior.
  4. Does const freeze objects? No. It prevents reassignment, not mutation.
  5. How to check NaN? Use Number.isNaN(value).
